GitHub Card

Embed beautiful, responsive GitHub repository cards in your Jekyll site with a simple Liquid tag.

v0.2.0 Jekyll Plugin Light & Dark

Quick Start

Use the simple Liquid tag to embed any GitHub repository.

Liquid
{% raw %}{% github r0x0d/jekyll-github-card %}{% endraw %}
{% github r0x0d/jekyll-github-card %}

Popular Repositories

Works beautifully with any public GitHub repository.

{% github facebook/react %} {% github microsoft/vscode %} {% github torvalds/linux %} {% github rails/rails %}

Multi-Language Support

Automatic language detection with color-coded badges.

{% github python/cpython %} {% github golang/go %} {% github rust-lang/rust %}

Features

Theme Aware

Automatically adapts to light and dark themes using CSS media queries.

Live API Data

Fetches real-time stats from GitHub API including stars, forks, and language.

Responsive

Fluid design that looks great on desktop, tablet, and mobile devices.

Installation

Add the plugin to your Jekyll site in minutes.

Gemfile
gem 'jekyll-github-card'
_config.yml
plugins:
- jekyll-github-card
Layout HTML
<link rel="stylesheet" href="{{ '/assets/css/github-card.css' | relative_url }}">
